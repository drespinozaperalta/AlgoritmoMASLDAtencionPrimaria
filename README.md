# AlgoritmoMASLDAtencionPrimaria
🧠 Algoritmo Interactivo MASLD/MASH + Obesidad

Herramienta clínica interactiva basada en evidencia para el abordaje integral de pacientes con MASLD/MASH y obesidad, integrando diagnóstico, estratificación de fibrosis, fenotipos cardiometabólicos y selección de tratamiento farmacológico.

🚀 ¿Qué hace esta herramienta?

Convierte la evidencia en decisiones clínicas prácticas:

📊 Calcula automáticamente FIB-4
🔄 Clasifica riesgo de fibrosis (bajo, intermedio, alto)
🧪 Sugiere ruta diagnóstica (elastografía, seguimiento o referencia)
🧬 Identifica fenotipos clínicos relevantes
💊 Recomienda tratamiento farmacológico de obesidad basado en comorbilidades
⚖️ Calcula metas dinámicas de pérdida de peso (kg)
🧪 ¿Para quién es?
Endocrinólogos
Internistas
Médicos de atención primaria
Hepatólogos
Profesionales de salud con enfoque cardiometabólico
